## Personal Portfolio Website

A modern and responsive **personal portfolio website** built using **HTML, CSS, and JavaScript** to showcase my skills, projects, and educational background.  
The website includes well-structured sections, dynamic content rendering with JavaScript, and a clean user interface for smooth navigation. It highlights my development skills while presenting my work and achievements in a professional and visually appealing manner.

### Features
- Responsive design for different screen sizes
- Projects section with live demo and source code links
- Dynamic content rendering using JavaScript
- Clean and user-friendly interface
- Smooth navigation between sections

### Tech Stack
- **HTML5** – Structure  
- **CSS3** – Styling and responsiveness  
- **JavaScript** – Dynamic functionality
